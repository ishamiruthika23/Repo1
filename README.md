# Coffee Sales Prediction using Machine Learning

This repository contains a Machine Learning project that predicts coffee sales
based on historical sales data. The project demonstrates a complete ML pipeline
including preprocessing, feature selection, scaling, model training, and deployment.

---

## Project Overview
Sales prediction is an important application of Machine Learning in business
analytics. In this project, a trained ML model is used to predict coffee sales
by learning patterns from past data.

The trained model and preprocessing components are saved and reused for
making predictions through a Python application.

---

## Dataset
- Historical coffee sales dataset
- Contains numerical and categorical features related to sales
- Data is preprocessed before model training

---

## Methodology
1. Data loading and exploration  
2. Data preprocessing and cleaning  
3. Feature selection  
4. Feature scaling  
5. Model training  
6. Model evaluation  
7. Model serialization for deployment  

---

## Machine Learning Components
- Trained Sales Prediction Model (`coffee_sales_model.pkl`)
- Feature Selector (`feature_selector.pkl`)
- Scaler (`scaler.pkl`)

---

## Results
The trained model is able to predict coffee sales with good accuracy by learning
patterns from historical data, making it useful for basic sales forecasting tasks.

---

## How to Run

1. Clone the repository  
2. Navigate to the project folder  
3. Install required libraries:
   ```
   pip install -r requirements.txt
   ```
4. Run the application:
   ```
   python app.py
   ```
